# OS Self Learning

Code for the OS self-learning activity.

## Disk Scheduling (C)

This is a simple C program to demonstrate disk scheduling algorithms:

* SSTF
* LOOK
* C-LOOK

The program takes a request sequence and initial head position as input, then shows head movement and total seek time.

## Requirements

Uses:

* stdio.h
* conio.h
* dos.h

Works best on Turbo C / DOSBox.

## Run

Compile and run the program, then enter inputs as prompted.
